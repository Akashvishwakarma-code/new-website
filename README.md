## Features

1. **Fully Responsive Design**

   - Mobile-first approach
   - Tablet and desktop optimizations
   - Cross-browser compatibility

2. **Modern UI Components**

   - Animated navigation bar
   - Interactive hero section with statistics
   - Business cards with hover effects
   - Progress bars for sustainability metrics
   - Contact form with validation

3. **JavaScript Functionality**

   - Animated counters
   - Tab filtering for businesses
   - Mobile menu toggle
   - Smooth scrolling
   - Form submissions

4. **Performance Optimized**
   - Optimized images (using Unsplash)
   - Efficient CSS animations
   - Lazy loading for animations

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## How to Use

1. Clone or download the repository
2. Open `index.html` in your browser
3. No build process required - it's plain HTML/CSS/JS

## Customization

### Colors

Primary colors are defined in `css/style.css`:

- Blue: `#0066b3` (Tata Blue)
- Red: `#e4002b` (Tata Red)
- Orange: `#ff6b00` (Accent)

### Images

Replace Unsplash image URLs with your own:

- Hero background: Line 47 in `index.html`
- Business cards: Each card has its own image URL
- About section: Line 40 in about-section HTML

### Content

Edit the HTML file to update:

- Text content
- Statistics
- Business information
- Contact details
